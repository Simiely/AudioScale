# AudioScale

通过音频驱动 After Effects 图层缩放的 ExtendScript 面板脚本。

适用于 **After Effects 2026**，基于 ExtendScript 实现，无需编译，一键安装即可使用。中/英文版 AE 通用。

---

## 功能特性

- **音频 → 缩放**：自动调用 AE 内置 `Convert Audio to Keyframes`，把音频振幅烘焙为关键帧，再通过表达式映射到目标图层的 Scale 属性。
- **三种映射模式**：
  - **基础振幅映射**：`scale = base + amp × intensity`，线性映射，简单直接。
  - **平滑 + 阈值**：用 `Property.smooth()` 去抖动，并设底噪阈值过滤静音段，适合人声/鼓点。
  - **频段分离**：复制音频层并加 `Bass & Treble` 粗分低/中/高频，多个目标图层自动轮询分配到不同频段（如低频驱动文字、高频驱动粒子）。
- **ScriptUI 可停靠面板**：参数可视化调节，支持多目标图层批量处理。
- **完整 Undo 支持**：所有操作包裹在 `app.beginUndoGroup` 内，可一键撤销。
- **2D / 3D 图层通吃**：表达式用 `value` 关键字继承当前 Scale 维度，自动适配 2D `[x,y]` 与 3D `[x,y,z]`。
- **跨语言兼容**：表达式与 API 调用全部走索引 / matchName，不依赖本地化显示名。

## 效果

| 场景 | 模式 | 效果 |
|------|------|------|
| 音乐可视化 | 基础 | 图层随节拍整体放大缩小 |
| 人声口型/呼吸感 | 平滑+阈值 | 过滤底噪，缩放柔和不抖动 |
| 多元素律动 | 频段分离 | 低频驱动主标题，高频驱动粒子/光点 |

## 安装

1. 下载 `AudioScale.jsx`。
2. 复制到 AE 脚本目录：
   - **停靠面板**：`Scripts/ScriptUI Panels/`（推荐，可在 AE 内窗口停靠）
   - **普通窗口**：`Scripts/`
3. 重启 After Effects。
4. 菜单 **窗口** → **AudioScale.jsx** 打开面板。

---

## 使用方法

1. 打开一个合成。
2. **同时选中**：
   - 一个含音频的图层（音频源）
   - 一个或多个目标图层（要被缩放的对象，Ctrl/Shift 多选）
3. 在面板里选择模式并调整参数。
4. 点击 **"应用到选中图层"**。

脚本会自动生成 `Audio Amplitude` 图层并在目标层 Scale 上挂载表达式。

## 参数说明

### 通用参数（所有模式生效）

| 参数 | 默认值 | 作用 | 调节直觉 |
|------|--------|------|----------|
| **强度** | 30 | 振幅到缩放像素的放大系数。`s = amp × intensity` | 调大→缩放更夸张；调小→更平缓；设 0→只有基准值不响应音频 |
| **基础缩放** | 100 | 静音时的基准缩放（%）。`scale = baseScale + s` | 静音时图层停在 baseScale；音频只叠加增量 |

### 模式选择

| 模式 | 适用场景 |
|------|----------|
| **基础振幅映射** | 音乐可视化、节拍跟随，简单直接 |
| **平滑+阈值** | 人声、鼓点，需要去抖动和过滤底噪 |
| **频段分离** | 多元素律动，不同频段驱动不同图层 |

### "平滑+阈值"模式专属参数

| 参数 | 默认值 | 作用 | 调节直觉 |
|------|--------|------|----------|
| **平滑宽度** | 0.1 | 平滑滤波时间窗口（秒）。`ampP.smooth(smoothW, 5)` | 调大→更平滑但响应迟钝；调小→更锐利但易抖动 |
| **底噪阈值** | 8 | 噪声门限。低于此值的振幅视为 0 | 调大→只有较大声音才触发；调小→更灵敏连底噪也响应 |

### "频段分离"模式专属参数

| 参数 | 默认值 | 作用 | 调节直觉 |
|------|--------|------|----------|
| **频段数** | 3 | 把音频分成 2 或 3 个频段 | 2→低/高频；3→低/中/高频。目标图层按顺序轮询分配到各频段 |

## 缩放控制逻辑

核心公式（所有模式一致）：

```
最终缩放 = 基础缩放 + 振幅增量 × 强度
```

三种模式的区别只在**如何预处理振幅值**：

- **基础**：直接用原始振幅
- **平滑+阈值**：振幅 → smooth 滤波 → 阈值过滤 → 增量
- **频段分离**：复制音频层 → Bass & Treble 分频 → 各自转关键帧 → 多图层轮询驱动

表达式统一用 `value` 关键字继承 Scale 当前维度，2D/3D 图层自动适配：

```javascript
v = value;                 // 2D: [x,y]  3D: [x,y,z]
v[0] = baseScale + s;      // 只改 X
v[1] = baseScale + s;      // 只改 Y
v                          // 3D 层的 Z 保持原值
```

## 注意事项

- **中文版 AE 完全兼容**：表达式里所有效果/属性引用均使用索引（如 `effect(3)(1)`），不依赖本地化显示名。
- **菜单命令**：脚本已包含中英文菜单名候选。若仍报"找不到菜单"，请在 [convertAudioToKeyframes](AudioScale.jsx) 的 `names` 数组里补充实际菜单名。
- **频段分离精度**：使用 `Bass & Treble` 粗略分频。如需更精确，可改为 `Parametric EQ` 并设置频率/带宽。
- **3D 图层**：自动保留 Z 轴缩放原值，只驱动 X/Y。
- **重复运行**：每次运行会重新生成 Audio Amplitude 图层，建议运行前清理旧的同类图层。
