01 / Modes

三种模式，各司其职。

📈
基础振幅映射
scale = base + amp × intensity，线性映射。振幅越高缩放越大，简单直接，适合音乐可视化。
〰
平滑+阈值
Property.smooth() 去抖动 + 底噪阈值过滤静音段。缩放柔和不乱跳，适合人声/鼓点驱动。
↔
频段分离
Bass & Treble 粗分低/中/高频，目标图层轮询分配。低频驱动文字，高频驱动粒子，各跳动各的。
🎯
2D / 3D 通吃
表达式用 value 关键字继承当前维度。2D 图层改 [x,y]，3D 图层改 [x,y] 保留 [z]，自动适配。
🌐
跨语言兼容
表达式里效果/属性的引用全部走索引（effect(3)(1)），不依赖本地化显示名。中文/英文/日文版 AE 都可用，无需额外配置。
02 / Params

灵活掌控。

通用
强度 · Intensity
振幅→缩放像素的放大系数（默认 30）。调大伸缩更夸张，调小平缓过渡，设 0 则完全不响应音频。
通用
基础缩放 · BaseScale
静音时的基准缩放值（默认 100%）。设为 50 则静音时缩小到一半，响的时候才放大叠加。
平滑模式
平滑宽度 · SmoothW
AE 内置 smooth 滤波窗口（秒，默认 0.1）。越大越平滑迟钝，越小越锐利跟随紧。
平滑模式
底噪阈值 · Threshold
噪声门限（默认 8）。低于该值的振幅视为 0，高于则减去阈值再映射。越大只有强音触发，越小连底噪也响应。
频段模式
频段数 · BandCount
分频数量（2 或 3，默认 3）。2→Low/High，3→Low/Mid/High。目标图层按顺序轮询分配到各频段。
核心公式
最终缩放 · Result
最终缩放 = 基础缩放 + 振幅增量 × 强度。三种模式只改变"振幅增量"的预处理方式，内核公式一致。
03 / Usage

三步上手。

STEP 01
📁
放入目录
将 AudioScale.jsx 放入 AE ScriptUI Panels 目录
STEP 02
🪟
打开面板
AE 窗口菜单 → AudioScale.jsx，开启控制面板
STEP 03
✨
一键应用
选中音频+目标图层 → 设模式调参数 → 点击应用，自动生成振幅关键帧与缩放表达式
运行后表达式自动挂载在工作区，不再依赖脚本。支持 Ctrl+Z 整体撤销。
